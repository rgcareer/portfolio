# Lessons — portfolio-site

[2026-06-23] | Shipped a static, motion-less "Phase A" as a premium *milestone* when Ryan's
explicit #1 priority was mind-blowing motion/animation/interactivity. He found it boring vs. the
earlier interactive mockups (command palette, particle motion, magnetic buttons, typing). | Rule:
when the user's stated top want is motion/wow, build the motion into the FIRST visible iteration —
never present a motion-less intermediate as a showcase. If a static scaffold must be shown, label it
scaffold and lead with the interactive layer. See memory [[feedback_motion-first-wow]].
